Algo Compare

×

Algo Compare, an algorithm comparison application. Users choose the number of times they want a random number generation algorithm to shuffle a deck of cards using the Fisher-Yates shuffle, and the amount of time that the task took is output. Currently the only number generating algos that you can test are the inbuilt Javascript Math.random and Xorshift. I'm interested in graphing the output in the future instead of just displaying the results in text.
